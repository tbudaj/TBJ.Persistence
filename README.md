# TBJ.Persistence

Repository containing NuGet packages for the persistence layer used in TBJ projects.

## Available packages

| Package | Description | Documentation |
|---|---|---|
| `TBJ.Persistence.EfCore` | Generic Repository and Unit of Work pattern for Entity Framework Core. Provider-agnostic, supports multi-tenancy. | [docs/TBJ.Persistence.EfCore/TBJ.Persistence.EfCore.md](docs/TBJ.Persistence.EfCore/TBJ.Persistence.EfCore.md) |

## Contributing

The project follows a Pull Request model. All changes require a review by the repository owner (`@tbudaj`).
CI/CD configuration, versioning details and local build instructions are in the individual package documentation.
